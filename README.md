# Algorithm and Data Structures (ALDA) - SmartStore

This repository contains the project implementation for the **Algorithms and Data Structures (ALDA)** course at **FH Campus Wien**. The objective of this project is to implement core computer science data structures and algorithms from scratch, explicitly avoiding the use of the built-in Java Collections Framework.

##  Project Overview

The **SmartStore** application serves as a practical environment to demonstrate custom-built data structures. It includes components like a product management system (`ProductTree`) and analytical dashboards, fully verified through unit testing.

### Key Features
* **Custom Binary Search Tree (BST):** Implements core BST structures and lookup algorithms for efficient data retrieval.
* **Hash Tables:** Custom collision-handling mechanisms implemented without native Java utility wrappers.
* **Quicksort Algorithm:** Custom sorting array manipulation optimized for performance.
* **Interactive Dashboard:** Features a welcome header and an analytics interface for milestone tracking.
